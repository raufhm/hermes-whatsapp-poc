# Hermes WhatsApp PoC

A proof-of-concept implementation for WhatsApp integration.

## Overview

This project explores WhatsApp messaging capabilities and integration patterns.

## Getting Started

### Prerequisites
- Go 1.16+

### Installation

```bash
git clone <repository-url>
cd hermes-whatsapp-poc
go mod download
```

### Usage

```bash
go run main.go
```

## Project Structure

- `main.go` - Entry point
- `go.mod` - Module dependencies
